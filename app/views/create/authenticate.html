Authenticating using a government gateway ID

In order to use this service you will need to authenticate who you are by using a government gateway ID.

If you already have a government gateway user ID you will be prompted to sign in.

If you do not, you will need to set up a user ID.

To set up a user ID as an individual you will need to provide:

  • your name and email address

To set up a user ID as a company you will need to provide:

  • your company name and company email address
